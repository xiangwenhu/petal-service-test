## [petal-service](https://github.com/xiangwenhu/petal-service) 测试项目


### 本地测试流程
1. 把本项目和petal-service 放在同一目录
2. ```npm  install```
4. 运行


### 本地搭建流程
1. 把本项目和petal-service 放在同一目录
2. `npm install ../light-service-master`
3. 查看 package.json
```json
  "dependencies": {
    "petal-service": "file:../light-service-master"
  },
```
4. 运行
